# `jniLibs/x86_64/` - emulator model library

This directory is the Android x86_64 output location for:

```text
libLlama-3.2-3B-Instruct-q4f16_0-MLC.so
```

The file is generated by Gradle task `buildModelLibSo` from:

```text
app/src/main/assets/Llama-3.2-3B-Instruct-q4f16_0-MLC-android-x86_64.tar
mlc4j/src/main/jniLibs/x86_64/libtvm4j_runtime_packed.so
```

Build the x86_64 model tar and matching TVM FFI-capable runtime first:

```bash
bash scripts/install_mlc_llm.sh
TARGET_ABI=x86_64 bash scripts/compile_model_tar.sh
TARGET_ABI=x86_64 bash scripts/fetch_mlc_native.sh
```

The generated `.so` files are ignored by git because they are machine-built
native artifacts.
